## üöÄ Getting Started

### Prerequisites
- Flutter SDK 3.4.0+
- Firebase account
- API keys for external services

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/coinpath.git

# Install dependencies
cd coinpath
flutter pub get

# Set up Firebase (follow instructions in pubspec.yaml)

# Run the app
flutter run
```

### Firebase Setup
1. Create a Firebase project at https://console.firebase.google.com/
2. Enable Authentication (Email/Password)
3. Create Firestore database
4. Add your configuration files as detailed in pubspec.yaml
